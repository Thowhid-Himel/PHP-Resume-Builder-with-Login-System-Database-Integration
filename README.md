A full-stack Resume Builder Web Application built using PHP, MySQL, HTML, Tailwind CSS, and JavaScript. It allows users to create, customize, save, and manage professional resumes with live preview and multiple templates.

🚀 Features
🔐 User authentication (Login / Register)
🧑‍💼 Resume dashboard
📄 Create, edit, delete resumes
🎨 6 different resume templates
👀 Live preview editor
🖼️ Profile picture upload
📂 Resume gallery system
💾 Save resumes in MySQL database
🖨️ Print / Export as PDF
⚙️ Show/Hide sections (Skills, Experience, Education, Summary)
🛠️ Tech Stack
Frontend: HTML, Tailwind CSS, JavaScript
Backend: PHP (PDO)
Database: MySQL
Other: AJAX Fetch API, Session Management
